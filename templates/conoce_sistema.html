Conoce nuestro sistema

Plataforma de autenticación y protección de usuarios

🔐

Sistema de seguridad

Nuestro sistema fue creado para proteger cuentas mediante validación de identidad, contraseñas seguras y métodos de recuperación.

👤

Registro de usuarios

Al crear una cuenta se almacena información necesaria para identificar al usuario y permitir un acceso personalizado.

  • Usuario
  • Correo electrónico
  • Teléfono
  • Categoría de seguridad
  • Palabra secreta
🛡️

Protección de contraseña

Las contraseñas no se guardan directamente. El sistema utiliza protección mediante hash para evitar que puedan ser recuperadas por terceros.

🔑

Recuperación de acceso

Si el usuario pierde el acceso puede utilizar diferentes métodos de seguridad:

  • Llave maestra
  • Palabra de seguridad
  • Sistema de verificación
📱

Verificación avanzada

El sistema puede enviar códigos de seguridad mediante correo electrónico o mensajería para confirmar la identidad.

⚙️

Tecnologías utilizadas

El proyecto utiliza:

  • Python + Flask
  • SQLite
  • HTML, CSS y JavaScript
  • Sistemas de autenticación segura
