Términos y Condiciones de Uso

Última actualización: 18 de junio de 2026

Al crear una cuenta en nuestra plataforma, el usuario acepta los siguientes términos y condiciones relacionados con el uso del sistema, seguridad y tratamiento de información personal.

1. Registro de usuario

Para crear una cuenta, el sistema puede solicitar información necesaria para brindar los servicios de autenticación y seguridad, incluyendo:

  • Nombre de usuario.
  • Contraseña.
  • Correo electrónico.
  • Número telefónico.
  • Categoría y palabra de seguridad elegida.
  • Información técnica del dispositivo utilizado.

2. Uso de la información recopilada

Los datos registrados serán utilizados únicamente para:

  • Crear y administrar la cuenta.
  • Permitir inicio de sesión seguro.
  • Verificar identidad.
  • Enviar códigos mediante correo o WhatsApp.
  • Detectar accesos sospechosos.
  • Mejorar la seguridad.

3. Datos técnicos de seguridad

El sistema puede almacenar información como:

  • Dirección IP.
  • Ubicación aproximada.
  • Fecha y hora de registro.
  • Sistema operativo o dispositivo.

4. Protección de contraseñas

Las contraseñas no se almacenan en texto visible. El sistema utiliza métodos seguros de protección mediante hash.

5. Llaves y códigos de seguridad

El sistema puede generar códigos temporales y llaves maestras para recuperación de cuenta. El usuario no debe compartir estos códigos.

6. Privacidad de datos

La información personal no será vendida ni compartida con terceros con fines comerciales. Solo será utilizada para funcionamiento, seguridad y recuperación.

7. Servicios externos

El sistema puede utilizar servicios externos de correo electrónico y mensajería para entregar códigos de seguridad.

8. Responsabilidad del usuario

  • Mantener segura su contraseña.
  • No compartir códigos.
  • Informar accesos sospechosos.
  • Usar correctamente la plataforma.

9. Seguridad del sistema

Se aplican medidas como:

  • Verificación por códigos.
  • Validación de contraseña.
  • Detección de dispositivos.
  • Recuperación segura.

10. Aceptación

Al registrarse, el usuario confirma que ha leído y acepta estos términos y condiciones. Si no está de acuerdo, no debe crear una cuenta.
